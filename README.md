# Zepto Sales Dashboard Analysis (Excel Project)

## Overview
This project is an interactive Excel dashboard created to analyze Zepto sales performance, outlet trends, ratings, and product categories.

The dashboard converts raw grocery sales data into actionable insights using pivot tables, slicers, KPIs, and charts.

## Features
- Interactive Excel Dashboard
- KPI Cards
- Pivot Tables
- Pivot Charts
- Dynamic Slicers
- Sales Analysis
- Outlet Performance Analysis
- Product Category Insights

## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Data Visualization

## KPIs Included
- Total Sales
- Average Sales
- Average Rating
- Number of Items

## Insights
- Outlet size performance
- Outlet location comparison
- Fat content analysis
- Item type sales trends
- Outlet establishment trends
